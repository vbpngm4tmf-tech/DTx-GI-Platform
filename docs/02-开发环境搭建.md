# DTx-GI 开发环境搭建

> **文档编号**：02  
> **版本**：V1.0

---

## 前置要求

| 工具 | 版本 | 说明 |
|------|------|------|
| Node.js | >=18 | 前端运行环境 |
| Go | >=1.21 | 后端运行环境 |
| Python | >=3.10 | AI开发环境 |
| Docker | >=24 | 容器化部署 |
| PostgreSQL | >=15 | 数据库 |
| Redis | >=7 | 缓存 |

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/vbpngm4tmf-tech/DTx-GI-Platform.git
cd DTx-GI-Platform

# 安装前端依赖
npm install

# 启动开发服务器
npm run dev
```

## 环境变量

```bash
# .env
cp .env.example .env

# 编辑 .env
DATABASE_URL=postgresql://user:pass@localhost:5432/dtxgi
REDIS_URL=redis://localhost:6379
JWT_SECRET=your-secret-key
```

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 端口冲突 | 修改 vite.config.ts 中的 port |
| 数据库连接失败 | 检查 DATABASE_URL 配置 |
| 权限不足 | 使用 sudo 或修改目录权限 |
