## 一、产品概述

### 1.1 产品定位

面向胃肠道疾病患者的数字疗法应用，提供个性化、全周期的术后康复管理服务。

### 1.2 核心价值

```
智能评估 -> 个性化方案 -> 康复训练 -> 效果追踪 -> 动态调整
```

---

## 二、功能模块

### 2.1 健康档案

| 功能 | 说明 |
|------|------|
| 基本信息管理 | 姓名、年龄、身高、体重、过敏史 |
| 病历记录 | 手术记录、病理报告、检查报告 |
| 用药记录 | 药物名称、剂量、服用时间 |

### 2.2 症状管理

| 功能 | 说明 |
|------|------|
| 症状记录 | 每日症状评分（疼痛、恶心、食欲等） |
| 智能预警 | AI识别异常症状模式，自动预警 |
| 医生沟通 | 与主治医生在线沟通 |

### 2.3 康复训练

| 功能 | 说明 |
|------|------|
| 运动指导 | 术后康复运动视频教程 |
| 饮食管理 | 个性化营养方案、食谱推荐 |
| 心理支持 | 心理健康评估、放松训练 |

### 2.4 随访管理

| 功能 | 说明 |
|------|------|
| 随访提醒 | 定期复查提醒 |
| 问卷填写 | 生活质量评估问卷 |
| 报告查看 | 检查报告在线查看 |

---

## 三、数据接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/symptom | POST | 提交症状记录 |
| /api/v1/training | GET | 获取训练计划 |
| /api/v1/followup | GET | 获取随访信息 |
| /api/v1/report | GET | 获取检查报告 |
