## 一、备案政策

根据工信部规定，微信小程序需完成备案后方可上架。

| 项目 | 要求 |
|------|------|
| 备案时限 | 上架前完成 |
| 备案主体 | 小程序运营主体 |
| 前置条件 | 需先完成ICP备案 |

---

## 二、材料清单

| 序号 | 材料名称 | 说明 |
|------|----------|------|
| 1 | 主体证件 | 事业单位法人证书 |
| 2 | 负责人身份证 | 正反面照片 |
| 3 | 小程序信息 | 名称、简介、服务类目 |
| 4 | 承诺书 | 备案真实性承诺书 |
| 5 | 前置审批 | 如涉及医疗需卫健委审批 |

---

## 三、备案流程

```
1. 准备ICP备案（或确认已有）
2. 登录微信公众平台
3. 填写备案信息
4. 上传材料
5. 平台初审（1-3工作日）
6. 管局审核（20工作日）
7. 备案完成
```
