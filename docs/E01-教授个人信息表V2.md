## 四、主要研究方向

1. **胃肠道肿瘤微创治疗** - 机器人辅助胃肠道肿瘤根治术、腹腔镜胃癌D2根治术
2. **胃肠道间质瘤（GIST）综合诊疗** - 分子分型与精准治疗、耐药机制研究
3. **数字疗法与人工智能在胃肠外科的应用** - DTx-GI平台研发、AI辅助诊断
4. **胃肠外科手术加速康复（ERAS）** - 围手术期营养支持、患者报告结局（PROs）
