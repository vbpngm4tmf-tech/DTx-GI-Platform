# DTx-GI 部署指南

> **文档编号**：03  
> **版本**：V1.0

---

## 一、Cloudflare Pages 部署

### 1.1 自动部署（推荐）

代码推送到 `main` 分支后，GitHub Actions 自动部署到 Cloudflare Pages。

### 1.2 手动部署

```bash
# 构建
npm run build

# 使用 Wrangler 部署
npx wrangler pages deploy dist --project-name=dtx-gi-platform
```

## 二、环境配置

| 环境 | 分支 | URL |
|------|------|-----|
| 开发 | dev | https://dev.dtx-gi.pages.dev |
| 预发布 | staging | https://staging.dtx-gi.pages.dev |
| 生产 | main | https://dtx-gi.pages.dev |

## 三、域名绑定

1. 在 Cloudflare Dashboard 添加自定义域名
2. 配置 DNS CNAME 记录指向 Pages 域名
3. 启用 SSL/TLS（自动）

## 四、环境变量

在 Cloudflare Dashboard 设置以下环境变量：

| 变量名 | 说明 |
|--------|------|
| DATABASE_URL | PostgreSQL 连接字符串 |
| REDIS_URL | Redis 连接字符串 |
| JWT_SECRET | JWT 签名密钥 |
| API_BASE_URL | API 服务地址 |
