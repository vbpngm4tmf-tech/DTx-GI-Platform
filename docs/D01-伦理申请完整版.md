## 第一部分 研究方案摘要

### 一、研究基本信息

| 项目 | 内容 |
|------|------|
| **研究名称** | DTx-GI 胃肠道疾病数字疗法平台研发与临床应用研究 |
| **研究负责人** | 洪流 教授 |
| **研究类型** | 干预性研究/前瞻性队列研究 |
| **研究周期** | 2025年3月 — 2028年3月（36个月） |
| **目标样本量** | 3,000例（干预组1,500例，对照组1,500例） |

### 二、研究目的

**主要目的**：评估 DTx-GI 平台对胃肠道肿瘤术后患者康复质量和生活质量的改善效果。

**次要目的**：
1. 评估平台对患者依从性和自我管理能力的影响
2. 评估AI辅助功能对症状识别和预警的准确性
3. 评估平台在降低术后并发症发生率和再入院率方面的效果

### 三、研究设计

多中心、随机对照、开放标签临床试验。受试者按1:1比例随机分配至干预组（DTx-GI平台+标准治疗）和对照组（标准治疗）。

## 第二部分 受试者保护

### 一、入选标准

1. 年龄18-75岁
2. 经病理确诊的胃癌或结直肠癌
3. 接受根治性手术切除
4. 术后分期I-III期
5. 能够使用智能手机
6. 签署知情同意书

### 二、排除标准

1. 存在严重认知障碍
2. 合并其他恶性肿瘤
3. 预期生存期<6个月
4. 正在参与其他干预性临床试验

### 三、知情同意

所有受试者在参与前必须签署书面知情同意书。知情同意书包含：
- 研究目的和过程
- 潜在风险和受益
- 自愿参与和随时退出权利
- 隐私保护措施
- 研究者联系方式

## 第三部分 数据安全计划

### 一、数据采集

- 所有数据通过加密通道传输
- 采用去标识化存储
- 访问权限分级管理

### 二、数据存储

- 数据库AES-256加密
- 每日自动备份
- 异地容灾

### 三、数据使用

- 仅用于本研究目的
- 不向第三方共享
- 研究结束后数据归档保存5年
